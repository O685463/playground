# AI Background Remover

A premium, browser-based tool to remove image backgrounds instantly using AI.

## Features
- **100% Client-Side**: No images are uploaded to any server. Privacy first.
- **AI Powered**: Uses advanced machine learning models via WebAssembly.
- **Free & Unlimited**: No credits, no sign-ups.
- **High Quality**: Supports high-resolution output.

## How to Use
1. Open the [Live Demo](https://o685463.github.io/playground/) (Replace with your actual URL).
2. Drag & drop an image or click to upload.
3. Wait a few seconds for the AI to process.
4. Download your transparent PNG!

## Development
This project uses:
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- [@imgly/background-removal](https://github.com/imgly/background-removal-js)

To run locally:
1. Clone the repository.
2. Open `index.html` in your browser.
   - *Note: Due to CORS restrictions with WebAssembly, you might need to run a local server (e.g., `python -m http.server` or VS Code Live Server).*

## Deployment
This project is ready for **GitHub Pages**.
1. Push to the `main` branch.
2. Go to Repository Settings > Pages.
3. Select `main` branch and `/ (root)` folder.
4. Save.
