# FBX Model Viewer

FBXファイルをブラウザ上で表示し、アニメーションを再生できるインタラクティブ3Dビューアです。

## 機能

- **FBXファイル読み込み**: ローカルから`.fbx`ファイルをドラッグ＆ドロップまたはクリックでアップロード
- **3Dビューア**: Three.jsによるリアルタイムレンダリング
- **カメラ操作**: 
  - マウスドラッグで回転  
  - スクロールでズーム
  - 右クリックで平行移動
- **アニメーション再生**: FBXに含まれるアニメーションの再生/一時停止
- **モデル情報表示**: 頂点数、マテリアル数、アニメーション数などの情報を表示

## 使い方

1. [FBX Model Viewer](../index.html)のカードをクリック
2. FBXファイルをドラッグ＆ドロップ、またはクリックしてアップロード
3. 3Dビューアでモデルを確認
4. マウス操作でカメラを動かして自由に閲覧
5. アニメーションがある場合は、再生/一時停止ボタンで制御

## 技術スタック

- **HTML/CSS**: バニラCSS + ガラスモーフィズムデザイン
- **JavaScript**: ES6モジュール
- **3Dライブラリ**: Three.js (CDN)
- **ローダー**: FBXLoader (Three.js公式)

## 開発

ローカルサーバで実行してください:

```bash
python -m http.server 8000
```

その後、`http://localhost:8000/tools/fbx-viewer/`にアクセス

## 注意事項

- ブラウザのセキュリティ制限により、ローカルサーバでの実行が必要です
- 大容量のFBXファイルは読み込みに時間がかかる場合があります
- WebGL対応ブラウザが必要です
