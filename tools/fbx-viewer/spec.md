# FBX Model Viewer 仕様書

## 概要
3D FBX ファイルをブラウザ上で表示し、モデルの回転・ズーム・アニメーション再生を可能にするインタラクティブビューアです。現在はプレースホルダー実装ですが、将来的に以下の機能を提供します。

## 主な機能
- **FBX ファイルの読み込み**: ユーザーがローカルから `.fbx` ファイルを選択またはドラッグ＆ドロップで読み込めます。
- **3D ビューア**: Three.js を利用し、モデルをリアルタイムでレンダリングします。
- **カメラ操作**: マウスドラッグで回転、スクロールでズーム、右クリックで平行移動が可能です。
- **アニメーション再生**: FBX に含まれるアニメーションをタイムラインで再生・停止・シークできます。
- **マテリアル表示**: テクスチャやマテリアル情報を保持し、適切に表示します。

## UI デザイン
- **カード表示**: `index.html` のツールグリッドに「FBX Model Viewer」カードを追加し、ステータスは `Available` と表示します。
- **ビューアページ**: `tools/fbx-viewer/index.html` に全画面キャンバスを配置し、上部にファイル選択ボタンとコントロールパネルを配置します。
- **デザイン指針**: 既存ツールと同様にガラスモーフィズムとパステル/アースカラーを使用し、プレミアム感を演出します。

## 技術スタック
- **HTML / CSS**: 既存のスタイルガイドに従い、`style.css` に共通クラスを利用。
- **JavaScript**: Vanilla JS + Three.js (CDN) を使用。モジュール化し、`script.js` にロジックを実装。
- **依存関係**: `three.min.js` (CDN) と `FBXLoader.js` (Three.js の公式ローダー) をインクルード。

## 今後の拡張案
- **GLTF/GLB 対応**: FBX 以外のフォーマットもサポート。
- **エクスポート機能**: 編集したモデルを GLTF 形式でエクスポート。
- **VR モード**: WebXR 対応で VR ヘッドセットで閲覧可能に。

## 開発手順 (簡易版)
1. `tools/fbx-viewer/` ディレクトリを作成し、`index.html` と `script.js` を配置。
2. `index.html` に Three.js CDN と FBXLoader のスクリプトタグを追加。
3. `script.js` でファイル入力ハンドラ、シーン・カメラ・レンダラーの初期化、アニメーションループを実装。
4. `style.css` にカード用クラスとビューアページ用のレイアウトを追加。
5. `index.html` のカードリンクを `tools/fbx-viewer/index.html` に設定し、ステータスバッジを `active` に変更。

## 注意点
- ローカルファイルの読み込みはブラウザのセキュリティ制限により、ローカルサーバ (`python -m http.server`) で実行する必要があります。
- 大容量 FBX ファイルはロードに時間がかかるため、ロード中のスピナー表示を検討してください。
